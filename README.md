AIO

Agenda inteligente para la gestión eficiente de objetivos personales

Versión 1.1:
- El sistema deberá ser compatible con múltiples sistemas operativos y dispositivos.
- Un usuario podrá registrarse.
- El sistema no deberá registrar a un usuario con un nombre de usuario mayor a diez caracteres.
- El sistema deberá obligar al usuario a registrarse con una contraseña que cumpla las condiciones establecidas.
- El sistema deberá aplicar un algoritmo de hash a la contraseña del usuario.
- Un usuario podrá iniciar sesión.
- El sistema deberá permitir al usuario iniciar sesión mediante el nombre de usuario o el correo electrónico junto con la contraseña.
- Un usuario podrá crear un objetivo.
- Un usuario podrá editar un objetivo.
- Un usuario podrá eliminar un objetivo.
- Un usuario podrá crear una tarea.
- Un usuario podrá editar una tarea.
- Un usuario podrá eliminar una tarea.
- Un usuario podrá añadir una actividad.
- Un usuario podrá añadir una rutina.
- Un usuario podrá cerrar su sesión.
- Un usuario podrá eliminar su cuenta.
- Un usuario podrá eliminar las notificaciones de su bandeja de entrada.
- Un usuario podrá vaciar su bandeja de entrada.
- Un usuario podrá visualizar la planificación semanal en su agenda.
- Un usuario podrá eliminar una actividad.
- Un usuario podrá eliminar una rutina.
- El sistema deberá emitir notificaciones a la bandeja de entrada del usuario.
- El sistema mostrará una página de error con información relevante.
- Un usuario podrá cambiar el color de su agenda.
- El sistema solo deberá permitir acceder a las funcionalidades a aquellos usuarios que hayan iniciado sesión previamente.
