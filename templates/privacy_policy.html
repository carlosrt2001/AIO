¿Qué datos recopilamos?

Los datos personales que recopilamos no son obligatorios. Éstos son:
-Tu correo electrónico.
-Tu número de teléfono.
-Las imágenes que cargas.

¿Para qué recopilamos estos datos?

Tu correo electrónico lo podrás utilizar para iniciar sesión en caso de haber olvidado tu nombre de usuario. Además, si introduces el correo electrónico, podrás iniciar el proceso para recuperar tu contraseña en caso de haberla olvidado o querer modificarla. Por último, podrás usar tanto el correo como el número de teléfono para exportar tu agenda via email o whatsapp.

Por su parte, las imágenes que subas como foto de perfil o imagen de objetivo al crear un objetivo, se utilizarán simplemente para personalizar tu experiencia en esta web.

Conservación de los datos

Los datos aportados por un usuario se conservarán indefinidamente en la base de datos de la aplicación hasta que la cuenta sea eliminada. En ese momento, todos los datos del usuario serán borrados permanentemente.

Derechos

Todos los usuarios tendrán los siguientes derechos:
-Derecho de acceso.
-Derecho de rectificación.
-Derecho de supresión.
-Derecho a la limitación del tratamiento.
-Derecho a la portabilidad.
-Derecho de oposición.

Para ejercer cualquiera de estos derechos, el usuario podrá solicitarlo enviando una solicitud a infoagenda.aio@gmail.com

Medidas de seguridad

Se han adoptado todas las medidas de seguridad posibles para tratar de garantizar la seguridad de todos los datos de un usuario almacenados en nuestra base de datos.

Cambios en la Política de Privacidad

La Política de Privacidad de esta aplicación web podrá ser modificada para adaptarse a cambios en la legislación o por la mejora o cambio en nuestro servicio. En caso de existir un cambio, éste se notificará a través de este sitio web.
